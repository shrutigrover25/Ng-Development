CSS SYNTAX :

Consists of a selector, property and its value 
Selector --> Points to the elements where the css is to be applied 
CSS Property is separated by semi colons 
Property:value 

selector {property : value }

h1{color:red}

p{color: red, font-size: 12px, padding: 1px};

P-> selector 
color, font-size, padding --> property 
red, 12px, 1px --> value

Declaration : A combination of a property and its corresponding value 

Selector : Used to target or select the elements on which I want to apply styles 

Property : specific characteristics of the element 

value : Assigned parameter

CSS ANIMATIONS :

change the behaviour of various elements in a web page.

@keyframes: specify the animation rule
animation-name: name 
animation-duration : time dusration it takes to complete one cycle
animation-timing-function: linear, ease-in, ease-out 
animation-delay : delay of the start of an element
animation=iteration-count: number of times the animation will be repeated 
animation-direction : direction of the animation 
animation-fill-mode: how styles are applied before and after the animation 
animation-play-state: whether the animation is running or is paused
